# tokei-tui

## 用法
`tokei . -o json | tokei-tui`

## 快捷键
| 快捷键 | 功能                       |
|--------|----------------------------|
| f      | 切换显示文件列表（-f）     |
| c      | 切换显示内嵌语言（-c）     |
| j      | 选择下一个文件             |
| J      | 选择最下方的文件           |
| K      | 选择最上方的文件(当前目录) |
| k      | 选择上一个文件             |
| h      | 返回上级目录               |
| l      | 进入下级目录               |
| Ctrl-f | 向下翻页                   |
| Ctrl-b | 向上翻页                   |
| g      | 翻到最上                   |
| G      | 翻到最下                   |
| q      | 退出                       |
